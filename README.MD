# Description
This is a CLI tool for encyrpting and uploading files to the IPFS blockchain network. The tool also generates unique file signatures and records them in a local database for one to keep track of the files.

# Planned / Additional Features
- Better error handling.
- File handling.
- Intraplanary FIlesystem Interface (IPFS) to upload and store data securley.
- CLI / Management Interface to keep track of you're files. 
- API Interface to easily view storage metadata information and query attributes of specific files.
